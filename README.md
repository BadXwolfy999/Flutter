# Bulk Contact Manager

A simple web application for adding and managing contacts in bulk with an easy-to-use interface.

## Features

- **CSV Upload**: Import multiple contacts at once via CSV file
- **Manual Entry**: Add contacts one by one with a simple form
- **Drag & Drop**: Easily upload CSV files with drag and drop functionality
- **Search & Filter**: Quickly find contacts by name, email, phone, or group
- **Group Management**: Organize contacts into custom groups
- **Local Storage**: Contacts are saved in your browser's local storage
- **Responsive Design**: Works on desktop and mobile devices

## Usage

### CSV Upload

1. Click on the "CSV Upload" tab
2. Drag and drop your CSV file or click to browse files
3. The application will automatically parse and import valid contacts
4. Download a template CSV file by clicking "Download CSV Template"

### Manual Entry

1. Click on the "Manual Entry" tab
2. Fill in the contact details (name, email, phone, group)
3. Click "Add Another Contact" to add more entries
4. Click "Save Contacts" to save all valid entries

### Managing Contacts

- **Search**: Type in the search box to filter contacts
- **Filter by Group**: Use the dropdown to show contacts from a specific group
- **Edit**: Click the edit icon to modify a contact
- **Delete**: Click the delete icon to remove a contact
- **Clear All**: Remove all contacts with the "Clear All" button

## Technical Details

- Built with vanilla JavaScript, HTML, and CSS
- No external libraries or frameworks required
- Uses browser's localStorage API for data persistence
- Implements responsive design with CSS Grid and Flexbox

## Installation

1. Clone this repository
2. Open `index.html` in your web browser
3. Start managing your contacts!
